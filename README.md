# Brent OS

Brent OS is a lightweight Arch-based gaming-focused Linux ISO.

## Features
- Arch Linux base (pacman)
- Systemd-free (dinit/openrc compatible)
- Gaming optimized (Steam, Proton, Vulkan)
- Minimal + debloated system
- DWM / lightweight WM setup

## Build ISO

Install dependencies:
```bash
sudo pacman -S archiso git
